# Zillow Seattle Page — Behaviors

## Navigation Bar
- **Sticky:** Stays fixed at top on scroll
- **Dropdown menus:** Each nav item (Buy, Rent, Sell, etc.) opens a dropdown on hover/click
- **Mobile:** Collapses to hamburger menu

## Search/Filter Bar
- **Sticky:** Pins below navigation on scroll
- **Tab switching:** For Sale / Rent / Sold tabs change the search context
- **Filter dropdowns:** Price, Beds & Baths, Home Type, More open filter panels
- **Search input:** Autocomplete suggestions on type

## Property Cards
- **Hover:** Subtle shadow increase, slight elevation
- **Image:** May have carousel dots for multiple photos
- **Heart/Save button:** Toggle save state on click
- **Status badges:** Colored labels (Showcase = blue, Open House = green, Price Cut = red)
- **Click:** Navigates to property detail page

## Pagination
- **Click:** Loads new page of results
- **Active page:** Highlighted with brand color

## Map (right panel)
- **Interactive:** Pan, zoom, click markers
- **Markers:** Show prices on map pins
- **Hover sync:** Hovering a listing highlights its map marker (and vice versa)
- **Responsive:** Hidden on mobile, toggle button to show

## Scroll Behavior
- Standard browser scrolling (no Lenis/Locomotive detected)
- Listings panel scrolls independently from map on desktop
